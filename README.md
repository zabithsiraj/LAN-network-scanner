# LAN scanner
This is purely bash sript used to scan your Local Area Network.   
# Installation 

    git clone https://github.com/zabithsiraj/LAN-scanner.git
# Usage 
    cd LAN-scanner
    
    ./network_scan.sh

# or make a command
    sudo mv networ_scan.sh /usr/bin/lan
# Then just type
    lan
